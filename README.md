# GRC-Core-Registers

Praktický Asset a Risk Register pro telemetrická IoT/OT řešení. Tento repozitář slouží jako ukázka implementace principů Governance, Risk & Compliance (GRC) podle ISO 27001 a NIS2.

## Obsah
- `Asset Register CORE.xlsx` – seznam všech aktiv s atributy (typ, kategorie, vlastník, umístění, stav, kritičnost, dostupnost, integrita, důvěrnost, síť, přístup, zálohování, aktualizace, poznámky).

## Použití
- Audit aktiv a rizik
- Plánování bezpečnostních kontrol
- Možné rozšíření o politiky, checklisty nebo monitoring
